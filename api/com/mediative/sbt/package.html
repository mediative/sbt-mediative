package sbt

Provides the following groups of sbt plugins:

  • Core - core settings and release plugins.
  • OSS - plugins for publishing and hosting open source software.
  • DevOps - plugins for packaging and delivering software into QA and production environments.
  • TeamCity - utilities for TeamCity builds.
